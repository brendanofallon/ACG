Download ACG

ACG runs on Mac, Windows, and Linux platforms. You don't need to configure, compile, make, or install ACG, just double-click the ACG icon.

Requirements

To run ACG you must have Java version 1.6 or later installed. If you're on a reasonably up to date Mac or Windows platform you're probably fine. On some Linux machines you may have to install it yourself, either through a package manager such as yum, yast or apt, or directly from Oracle

ACG can be something of a memory hog, especially when analyzing large or highly recombinant data sets. You'll probably be unhappy running ACG on a machine with less than 2GB of RAM. 4GB should be sufficient for all but the largest of runs.
